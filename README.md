# NoteNest 📝

NoteNest is a responsive notes management web application that helps users organize their notes inside folders. Each user has a private workspace, ensuring that folders and notes remain separate between different accounts on the same device.

## Features

* User Login & Sign Up
* Password Validation
* Show/Hide Password Option
* Create and Delete Folders
* Create, Edit, and Delete Notes
* Search Folders
* User-Specific Data Storage
* Home Navigation
* Logout Functionality
* Responsive Design
* Modern Yellow & Purple Theme
* Local Storage Data Persistence

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Local Storage

## How to Use

1. Create an account or log in.
2. Create folders to organize your notes.
3. Open a folder and add notes.
4. Edit or delete notes whenever needed.
5. Use the Home button to return to the folder dashboard.
6. Log out securely when finished.
